Nostr-Psbt Demo

A simple demo of coordinating PSBTs over Nostr.

Step 1: Send an invitation.

        * Send invitation to your pubkey to start a conversation.
        * Past conversation should be stored and viewable.
        * Conversations should be selectable and deletable.
        * Conversations should display a shared status.
        * Conversations should have labels for local organizing.
        * Conversations should be searchable by their attributes.
        * Nostr is a free place to store conversations (and fetch them later).
        
        * Two message types: chat messages and updates to the PSBT.
        * Diablo 2 style trade confirmations: All parties must check green,
          any changes to the contract cancels all agreements.
        * Once all parties agree, check is replaced by sign button.
        * Once enough parties sign (and fund), any party can broadcast.
